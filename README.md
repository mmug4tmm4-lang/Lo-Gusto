# نظام إدارة مطعم "لو غوستو" | Lo Gusto Restaurant Management System

مشروع برمي متكامل لتطوير نظام إدارة وطلب وجبات الطعام لمطعم "لو غوستو"، تم بناؤه لتسهيل عمليات الطلب وإدارة البيانات بكفاءة وسلاسة.

## إعداد وتطوير (Developed By)
* **عبد الله محمد خروب**

## التقنيات المستخدمة (Technologies Used)
* **PHP** (البرمجة الخلفية وإدارة الجلسات والمنطق البرمجي)
* **MySQL** (بناء وإدارة قواعد البيانات وجداول النظام)
* **HTML5 & CSS3** (تصميم وبناء الواجهات الأمامية للمستخدم)

## الميزات الرئيسية (Features)
* نظام تسجيل دخول وإدارة جلسات المستخدمين (User Sessions).
* قاعدة بيانات ديناميكية لعرض الوجبات، الأسعار، وإدارة الطلبات.
* واجهة مستخدم متجاوبة وسهلة الاستخدام وتدعم العمليات الأساسية للمطعم.
